# manifoldcontrol.github.io

the manifold control site: verification architecture for organizations deploying ai.
served at [manifoldcontrol.com](https://manifoldcontrol.com) via github pages.

static, no build step: `index.html` + `og.png`. project demos (e.g.
[/csr-seed/](https://manifoldcontrol.com/csr-seed/)) are published by their own repos
and serve under this domain automatically.
